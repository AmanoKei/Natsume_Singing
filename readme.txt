夏目悠里/日本語男声歌唱データベース（Natsume Yuuri/Japanese male singing database)について

■内容物
夏目悠里/日本語男声歌唱データベースは、権利的に問題ないポップス楽曲や
著作権の切れた同様をポップス調に歌い上げた歌唱データを収録したものです。

wav……………音声波形(48kHz/24bit)
mono_label……音素のモノフォンラベル情報
midi……………譜面のラベル情報
xml……………楽譜情報
list.xlsx………収録楽曲リスト

■声質

男性20代
爽やか

■収録機材

Audiotechnica AT2035
Audient iD4

■音素長

楽曲数……約50曲
pau,br,(cl)を除く音素区間……約48分
上記に加え、無声子音を除く音素区間………約42分


■音素ラベル付けについて

明治大学の東北きりたんDBを準拠して手動でラベリングしました。
また，無声子音の無音区間を無声子音側に入れています。

■発音記号について

Sinsyの辞書ファイルに準拠しました。
ただし、「を」については収録者の癖によって「wo」「o」の両方が混在しています。
ラベルは発音準拠、XMLはなるべく歌詞準拠で作成しています

※一例

歌詞「そうか」
発音「そおか」
音素「s/o/o/k/a」

■ノイズについて

ノイズ除去に関しては、音質を損なわないようになるべく不要な雑音を除去しました。
それ以外に何かしら致命的なノイズが発生している場合はお知らせください。

なお、12番については少々残響があるのでご留意ください。
